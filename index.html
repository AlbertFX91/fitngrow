Fit&Grow

Fit&Grow pretende ser una aplicación de monitorización de ejercicio físico para cualquier tipo de persona, deportista experto o principiante. La aplicación sirve, además, para llevar un registro del ejercicio físico realizado por el usuario, el cual deberá registrarse para mantener todo su historial dentro de la aplicación.

Este servicio de monitorización se complementará con un sistema de recompensas o gamificaciones dentro de la propia aplicación. Podrán conseguirse logros, medallas, desbloquear nuevos niveles de dificultad, conseguir nuevas rutas, así como ventajas dentro de la aplicación.

Fit&Grow contará además de un sistema de puntuación que se reconocerá como experiencia para el usuario y que se incrementará a medida que el usuario vaya evolucionando en su entrenamiento. Se contará con un plus especial para aquellas personas que decidan volver a entrenar después de un cierto tiempo, no llegando a ser muy elevado, ya que sería injusto con la puntuación por logros conseguidos por los demás usuarios.
